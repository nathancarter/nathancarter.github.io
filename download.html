To download the add-in, click here.

System Requirements:

  • Windows 7 Operating System
  • Microsoft Office Excel 2007
  • Visual Studio Tools for Office System 3.0 Runtime
To install the add-in, follow these instructions:
  1. Download the add-in using the above link.
  2. Specify the location where you would like the files to be saved (e.g. My Documents).
  3. Navigate to that location (e.g. if you selected My Documents, open My Computer, double-click C:\ drive, double-click Documents and Settings (XP) or Users (Vista/Windows 7), double-click your username, double-click My Documents.) There will be a folder called Koios – Excel Add-in. Double-click it.
  4. Double-click SETUP.EXE to begin the installation.
  5. Once the installation is complete, click Close.
